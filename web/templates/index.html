TTB Alcohol Label Verification

Automated compliance checking — 27 CFR §§ 5.32, 5.36, 5.121, 16.21 — fully offline.

How it works

Upload a label image and the expected field values from the COLA application. The system extracts text via Tesseract OCR, runs a trained CRF sequence tagger to identify each field (brand, class/type, ABV, net contents, government warning), then checks each field against 27 CFR requirements. A Markov chain flags anomalous brand names; an HMM forward pass scores overall label structure. Results return in under one second — no cloud API, no internet connection required.

Pipeline diagram 
Label image
  → GoCV: MSER crop · Otsu binarize · 2× upscale
  → Tesseract OCR  (--psm 4)
  → CRF tagger     (BIO: B-BRAND I-BRAND B-TYPE B-ABV B-NET B-WARN I-WARN)
  → HMM forward    (log-likelihood anomaly score)
  → Markov chain   (character bigram brand plausibility)
  → Comparison engine (fuzzy match · float tolerance · ALL-CAPS case check)
  → PASS / FAIL + per-field verdict JSON
