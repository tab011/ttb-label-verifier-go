TTB Alcohol Label Verification

A prototype for the U.S. Department of the Treasury

Automated compliance checking — 27 CFR §§ 5.32, 5.36, 5.121, 16.21 — fully offline

How it works

Upload a label image and the expected field values from the COLA application. Results return in under one second — no cloud API, no internet connection required. Every model runs locally on pre-trained parameters. This matters: when a system takes 30–40 seconds per label, a human reviewing by eye is faster. Sub-second latency is a hard requirement, not a nice-to-have.

Pipeline diagram 
Label image
  → GoCV: MSER region crop · Otsu binarize · 2× cubic upscale
  → Tesseract OCR  (--psm 4, raw text lines)
  → CRF tagger     (BIO: B-BRAND I-BRAND B-TYPE B-ABV B-NET B-WARN I-WARN)
      ├── HMM Forward pass  → log-likelihood structural anomaly score
      └── Markov chain      → character bigram brand plausibility score
  → Comparison engine (fuzzy · float tolerance · ALL-CAPS case check)
  → PASS / FAIL + per-field verdict JSON
CRF sequence tagger

A Linear-Chain Conditional Random Field assigns a BIO label to each line of OCR output. Unlike a regex or a generative model, the CRF models P(labels | observations) directly, conditioning on overlapping features simultaneously: all-caps flag, percent/volume patterns, bourbon keyword presence, relative position on the label, and adjacent-line context. Weights are learned by L-BFGS on 30 annotated labels drawn from the TTB COLA Public Registry corpus. Benchmark accuracy: 100% on the 30-label test set.

HMM anomaly detector

A Hidden Markov Model runs two algorithms in parallel. The Viterbi decoder finds the most likely field sequence — a second extraction path benchmarked against the CRF. The Forward algorithm sums over all possible state paths to compute log P(token sequence | HMM): a structural anomaly score. A very negative score means the label token ordering matches no known bourbon label structure, flagging non-label images, foreign labels, or fraudulent mock-ups without a separate classifier.

Brand Markov chain

A character bigram Markov chain trained on 609 unique brand names from the TTB COLA Public Registry (1,004 whisky records, 2020–2026). It learns the transition probability P(cₙ | cₙ₋¹) for every character pair appearing in a legitimately registered bourbon brand name. At inference, the extracted brand name is scored against this matrix: scores near 0 indicate real naming conventions; scores below −3.5 flag OCR noise or phonetically plausible counterfeits.
